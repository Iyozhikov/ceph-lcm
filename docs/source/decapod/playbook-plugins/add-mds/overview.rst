========
Overview
========

The following table shows the general information about the *Add
metadata server to the host* plugin:

====================    ===============================
Property                Value
====================    ===============================
ID                      ``add_mds``
Name                    Add metadata server to the host
Required Server List    Yes
====================    ===============================

following table shows the mapping between the plugin version and the
corresponding version of ``ceph-ansible``.
