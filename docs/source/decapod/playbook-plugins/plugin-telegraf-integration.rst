====================
Telegraf integration
====================

The *Telegraf integration* playbook plugin activates the Ceph metrics in
`Telegraf <https://www.influxdata.com/time-series-platform/telegraf/>`_. These
metrics can be sent to Prometheus, InfluxDB, or any other endpoint.
