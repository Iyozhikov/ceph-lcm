.. _plugins_add_client:

===================================
Add RBD and CLI clients to the host
===================================

This plugin adds required packages and config files to the node to
allow end user to execute :program:`ceph` tool. Also, it installs
:program:`rbd` and :program:`rados` tools and configures them.
