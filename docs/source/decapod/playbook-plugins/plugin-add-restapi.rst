.. _plugins_add_restapi:

========================
Add REST API to the host
========================

This plugin allows to deploy Ceph admin REST API to the host.

This REST API is not well documented, but you may
find some description on `Ceph Planet agregator
<https://ceph.com/geen-categorie/documentation-for-ceph-rest-api/>`_.
